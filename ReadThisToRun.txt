
To RUN the Angular APP
Step1 - Open "project-manager-mean" folder and go to root directory in cmd prompt.
Note: Skip Step2 to if node modules dependency files are already available.
Step2 - In CMD prompt type "npm install" to download all dependent node modules.
Step3 - Once node modules installed type "ng serve -o" in terminal.
Step4 - Now our application starts to run in the terminal mentioned localhost path.
